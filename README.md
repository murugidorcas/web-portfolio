

Welcome Page
This is a simple, visually appealing welcome page for a portfolio website. The page includes an animated introduction, a call-to-action (CTA) button that redirects to another page, and a navbar for easy navigation.
Features
Responsive Design: The layout adapts to various screen sizes, ensuring a smooth experience across devices.
Radial Gradient Background: A vibrant and dynamic background with a radial gradient effect, giving the page a modern, colorful feel.
Smooth Animations: The page content, including the welcome text, appears with a fade-in effect, adding a touch of interactivity.
Navbar: A fixed, centered navigation bar to access other sections or pages.
Interactive CTA Button: A clickable button that encourages users to explore the portfolio, with hover effects and smooth transition animations.
Technologies Used
HTML5: The structure of the webpage.
CSS3: Styling and animations to create a visually appealing experience.
JavaScript: A small function to handle overflow behavior for smooth page transitions.
How to Use
1.Clone the repository or download the HTML file.
2.Open index.html (or your main HTML file) in a browser to view the welcome page.
3.The Explore now button redirects to Homepage2.html when clicked. Ensure that you have Homepage2.html in your project directory.
Customization
You can easily customize the following:
Text: Modify the text inside the <h1 class="welcome-text"> tag to match your own welcome message.
Button Link: Update the onclick="location.href='Homepage2.html'" in the CTA button to redirect to a different page by changing the URL.
Colors: Modify the gradient background, text color, and button color in the CSS section to match your desire

Homepage2
This is a personal portfolio webpage that includes various sections to showcase different aspects of your profile. The layout features a fixed header, a side navigation bar, and animated effects to enhance the user experience.
Features
Fixed Header: A purple header that remains at the top of the page while scrolling.
Side Navigation: A vertical navigation bar that links to different sections of the portfolio, such as About, Interests, Projects, Education, Skills, and Contact.
Smooth Animations: Various elements on the page, including the header, navigation bar, and main content, fade or slide in with smooth animations when the page loads.
Responsive Design: The layout adapts to smaller screens, such as tablets and smartphones, with a reorganized navigation bar and adjusted font sizes.
Technologies Used
HTML5: The structure of the webpage, including the header, navigation bar, and main content sections.
CSS3: The styling of the webpage, including animations, transitions, and layout adjustments.
CSS Animations: Used to create engaging effects like fade-ins and slide-downs for the header and sections.
Media Queries: Ensures that the layout is responsive across devices, such as tablets and smartphones.
File Structure
index.html: The main HTML file for the portfolio page, which contains the header, navigation, and main content.
style.css: The CSS file containing all the styling, animations, and responsive media queries.
about.html, interests.html, projects.html, education.html, skills.html, contact.html: Other pages that should be linked in the navigation for a complete portfolio.
How to Use
1.Clone the repository or download the HTML and CSS files.
2.Open the index.html file in your browser to see the portfolio in action.
3.The navigation links will redirect to other sections like About, Projects, and more. Ensure these pages (e.g., about.html, projects.html) exist in the same directory for the links to work.
4.You can customize the content in each section (e.g., About, Projects) by editing the corresponding HTML files.
Customization
Header and Navigation Bar Colors: Modify the background colors of the header and navigation bar by adjusting the background-color properties in the CSS.
Font Style: Change the font family or font size by adjusting the font-family and font-size properties in the body or other text-related tags.
Animations: Adjust the duration or timing of the fade-in and slide-in animations by modifying the values of the animation properties in the CSS.
Responsive Design: The @media queries in the CSS file allow customization of the layout for different screen sizes. You can change the layout of the navigation or adjust font sizes by editing the media query values.
